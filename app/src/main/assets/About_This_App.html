关于此 APP

简介

ASRabbit是一个第三方晋江文学城论坛1 iOS 客户端, 提供优化移动端显示效果和简化使用逻辑的功能, 希望你喜欢（比干）。

功能

基本功能：

浏览版区、帖子、登录、回复、发帖、历史记录（MAX 998）、收藏帖子（MAX 998）。

增强功能：

  • 超(b)简(a)洁(d)的(t)显(a)示(s)效(t)果(e)；
  • 夜间模式（但是光线不好的环境下真的不要玩手机比较好...）；
  • 可定义版区顺序；
  • 滑动刷新 / 加载；
  • 压缩上传图片（GIF格式除外）；
  • 对应WLAN / WiFi模式自动开关的无图模式；
  • 可以选择使用 bbs.jjwxc.net / bbs.jjwxc.com 域名；
  • 记录浏览到的楼层数, 并在下次进入同一个帖子时直接跳跃到该楼层（如果上面有大量加载中的图片, 该功能可能会受影响）；
  • 使用层叠样式显示帖子内容；
  • 版区重复帖子过滤, 加载下一页时自动过滤前面出现过的帖子；
  • 帖子内点击不同楼层的右上角的回复按钮后, 自动复制该楼层文字到回复编辑框；
  • 超·强的富文本编辑功能, 你不仅能从相册/相机插入图片, 甚至可以从浏览器等地方复制图片（包括GIF！）直接粘贴到发帖/回复框2；
  • 多线程上传图片到图库并自动帮你完成贴图代码；
  • 搜索历史记录和收藏记录；
  • 帖子内有跳一楼和一键到底按钮;
  • 倒序阅读帖子内容;
  • 点击帖子内图片, 可以以漫画模式浏览帖子内所有已经加载的图片；
  • 按住帖子内图片可以保存或者复制图片(漫画模式下暂时不能这么干...)；
  • 过滤锁帖代码, 准确地说是过滤除了图片和超链接以外的所有HTML、JS、CSS代码；
  • 显示浏览过的帖子新增回复数；
  • 发帖、回复草稿功能, 最大草稿数量：1； (⊙﹏⊙)b
  • 简化顶置贴的显示效果；
  • 支持iPad Pro / iPad Air2上的分屏显示功能；
  • http://开头的链接可点击；
  • 支持超链接内部转跳；
  • 启动时自动清缓存。
  • 如果如果你有任何改进意见, 可以通过邮件联系我。

内容

  • 由于没有官方 API 的支持, 本 APP 以基于匹配晋江文学城论坛网页 HTML 文本内容进行显示。所以如果官方论坛改版的话, 大概需要大量的代码更新才能继续正常使用233。

声明

  • 这是一个因爱和兴趣而生的APP, 不会以任何理由向你索取费用。
  • 切记不要在任何非官方APP Store渠道下载使用该APP!
  • 本 APP 在使用过程中（浏览版区、帖子、图片、登录、发帖、回复等操作）会产生网络流量, 由此造成的费用需由用户自行承担。
  • 本 APP 将收集你的浏览历史并保存到手机内部储存, 仅用于显示历史记录。
  • 本 APP 将保存的你登录的账号密码及Cookies到手机内部储存, 仅用于登录界面的显示，还有回复及发帖认证。
  • 本 APP 仅提供优化移动端显示效果和简化使用逻辑的功能, 并不对用户浏览内容及输入信息承担任何责任。
  • 图片上传及外链服务由TieTuKu（贴图库）网站提供, 本 APP 仅为方便用户使用而接入其上传功能, 用户需要对自己上传的所有图片负责。
  • 本 APP 不会上传任何用户数据到晋江文学城论坛和TieTuKu（贴图库）以外的任何地方。
  • 本 APP 的所有头像源于杉森建（日文︰杉森すぎもり建けん Sugimori Ken）在Pokemon中创作的插画。
